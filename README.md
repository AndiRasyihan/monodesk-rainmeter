# MonoDesk — Tema Rainmeter Monokrom

Suite skin [Rainmeter](https://www.rainmeter.net/) serba hitam-putih yang bersih dan konsisten:
jam, statistik sistem, launcher aplikasi, to-do list, bingkai foto, dan pemutar musik.

## Skin yang Tersedia

| Skin | Fungsi |
| ---- | ------ |
| **Clock** | Jam + tanggal besar |
| **Stats** | CPU, RAM, disk, jaringan real-time |
| **Launcher** | 6 pintasan aplikasi dengan ikon monokrom otomatis |
| **TodoList** | Daftar tugas yang bisa dicentang |
| **Photo** | Bingkai foto dengan tab judul |
| **Music** | Kontrol & info pemutar musik |

## Fitur Khas

- **Zoom per skin** — scroll mouse di atas skin untuk membesar/mengecilkan (tersimpan otomatis)
- **Satu file konfigurasi** — ubah font, warna, dan daftar aplikasi di `@Resources/Variables.inc`, semua skin ikut berubah
- **Ikon launcher otomatis** — `make_icons.ps1` mengekstrak ikon aplikasi jadi versi monokrom
- **Wallpaper senada** — `make_wallpaper.py` menghasilkan wallpaper minimalis yang serasi

## Instalasi

1. Install [Rainmeter](https://www.rainmeter.net/)
2. Jalankan `install.bat` — skin tersalin dan aktif otomatis
3. (Opsional) Edit `MonoDesk/@Resources/Variables.inc` untuk kustomisasi, lalu jalankan ulang `install.bat`
